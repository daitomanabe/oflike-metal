# oflike-metal API Documentation

This directory contains detailed API documentation for oflike-metal.

## Structure

Documentation is organized by phase:

- `00_foundation.md` - Core classes (Context, Engine, AppBase)
- `01_math.md` - Math types (ofVec2f, ofVec3f, ofMatrix4x4, etc.)
- `02_graphics_2d.md` - 2D drawing (ofDrawCircle, ofDrawRectangle, etc.)
- `03_graphics_3d.md` - 3D drawing (ofDrawBox, ofDrawSphere, ofMesh, etc.)
- `04_color.md` - Color classes (ofColor, ofFloatColor)
- `05_image.md` - Image handling (ofImage, ofTexture, ofPixels)
- `06_camera.md` - Camera classes (ofCamera, ofEasyCam)
- `07_lighting.md` - Lighting (ofLight, ofMaterial)
- `08_fbo.md` - Framebuffer objects (ofFbo)
- `09_font.md` - Text rendering (ofTrueTypeFont)
- `10_events.md` - Event system (mouse, keyboard, window)
- `11_utils.md` - Utilities (string, file, log)

## API Compatibility

All APIs aim for 100% compatibility with openFrameworks. Any deviations are documented with rationale.

## Examples

Each API documentation includes:
- Function signatures
- Parameter descriptions
- Return values
- Usage examples
- Code snippets
