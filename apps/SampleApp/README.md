# SampleApp

Generated with oflike-gen (oflike-metal Project Generator).

## Build

### With XcodeGen (Recommended)

```bash
xcodegen generate
open SampleApp.xcodeproj
```

### With CMake

```bash
mkdir build && cd build
cmake .. -G Xcode
open SampleApp.xcodeproj
```
